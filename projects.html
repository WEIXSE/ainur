Детектор Фишинга

Задача: бинарная классификация текста (фишинг/не фишинг или спам/не спам) для входящих сообщений/писем.
Реализация: датасет → токенизация → обучение модели (baseline TF‑IDF+Linear/LogReg и нейросеть на PyTorch) → сравнение метрик → выбор лучшей модели.
Деплой: REST API (FastAPI) + Docker; endpoint /predict, логирование входов/выходов, скрипт для пакетной проверки.
Стек: Python, PyTorch, scikit‑learn, FastAPI, Docker.

ИИ для изображений

Задача: классификация изображений (например, “тип товара/дефект/категория” — выбирается под твою тему).
Реализация: обучение CNN (transfer learning: ResNet/EfficientNet), аугментации, confusion matrix, анализ ошибок.
Продуктовая часть: небольшой интерфейс (Streamlit/веб‑страница) для загрузки изображения и получения результата.
Стек: PyTorch, torchvision, OpenCV, Streamlit (или FastAPI).
/OPEN
